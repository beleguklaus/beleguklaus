About Our Server

JavaBool is a friendly Minecraft community where players can explore, build, and have fun together.

How to Connect

  1. Open Minecraft
  2. Go to Multiplayer
  3. Click "Add Server"
  4. Enter "Java Bool" as the server name
  5. Enter "mc.javabool.com" as the server address
  6. If connecting from a Java Client, use "48660" as server port
  7. If connecting from a Bedrock Client, use "48659" as server port
  8. Click "Done" and join the server!
